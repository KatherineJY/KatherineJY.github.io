Host Planning and Disk Partitioning

  1. Disk partition format: MBR (Master Boot Record) & GPT (GUID partition table).
  2. MBR Format
    • Master Boot Record, MBR (Store the boot manager, 446 bytes)
    • partition table (64 bytes) Since the partition table is only 64 bytes in capacity, there can be at most four groups of recording areas, and each group of recording areas records the starting and ending numbers of cylinder.
    • There can be at most one extension segmentation and the logical segmentation is a segmentation groove divided by it.
  3. GUID partition table
    • Logical Block Address, LBA
    • structure
      • LBA0 (MBR)
      • LBA1 (GBT header: position and size of partition table, position of backup GPT partition, check code)
      • LBA2-33 (starting and ending numbers of cylinder)
  4. BIOS
    • CMOS (A memory that records various hardware parameters and is embedded on the motherboard); BIOS (A software written to the motherboard)
    • process of turning up a computer
      1. BIOS
      2. MBR
      3. Boot Loader
      4. System core file
