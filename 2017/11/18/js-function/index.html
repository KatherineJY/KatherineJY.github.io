js Function类型

发表于 2017-11-18

摘录自《JavaScript高级程序设计》

  • 在js中，Function类型也是一个对象，函数名实际上是一个指向函数对象的指针

    三种定义函数的方法：

  • 函数名仅仅只是指向函数的指针，它不会影响这个函数对象。因此，在js中没有函数重载的概念，当声明了两个同名函数的时候，函数名指向了后一个声明的函数，从而丢弃前面的那个函数的指针。前面的那个函数对象仍然是存在的，而且没有被改变，所以只是函数指针的改变，并不能算是函数的重载。

  • 函数声明与函数表达式

    函数声明与函数表达式之间是存在区别的。在向执行环境中加载数据时，解析器会率先读取函数声明，并使其在执行任何代码之前可用；但是对于函数表达式，是当执行到它那里时，才会被解释执行。就是说，如果函数声明在调用它的语句之后，仍然能有效执行。但是函数表达式在调用它的语句之后的话，执行到调用语句时就会报错，找不到对应的函数。

  • 函数也可以作为值被当做另一个函数的参数被传递

  • 函数内部属性

    • arguments对象的callee属性，这个属性是一个指针，指向拥有这个arguments对象的函数

    • caller 这个属性中保存着调用当前函数的函数的引用。如果在全局作用域中掉用哪个当前函数，它的值为null。

  • 函数属性和方法

    • call() apply()都是在特定的作用域中调用函数。

      使用apply()、call()能够扩充函数赖以运行的作用域

    • bind() 这个方法会创建一个函数的实例，其this值会被绑定到传给bind()函数的值。
