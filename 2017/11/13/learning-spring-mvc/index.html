spring mvc入门

发表于 2017-11-13

本来是想从基础一点一点弄的，但是概念什么的完全搞不懂。所以就从最好上手的spring mvc开始了。
教程是慕课上的 spring mvc 起步

访问过程

  1. 用户通过浏览器发送request
  2. dispatcherServlet拦截request
  3. dispatcherServlet将查找相应Controller的操作代理给了HandlerMapping
  4. 找到了Controller和HandlerInterceptor。把Controller用HandlerInterceptor包装成一个Handler或者HandlerAdapter
  5. dispatcherServlet调用Handler
  6. Handler生成对应的模型，将模型ModelAndView返回给DispatcherServlet
  7. dispatcherServlet调用视图解析器,模型数据传给view对象
  8. 视图发回给浏览器
    除了Controller，其他都是可以通过配置完成
