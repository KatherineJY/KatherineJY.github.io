vps+ssr

发表于 2017-11-09

就是有一种执念，非要上google

Github Education

里面有个学生的discount package，很良心。而且申请超级快，隔天就申请到了，效率感人。提供的discount package里面就有digital ocean的50刀代金券。

digital ocean

  • vps是在digital ocean购买的，对的，就是为了用代金券。激活账号还是要花费5刀的。
  • 激活后选择一台服务器 我选的是ubuntu操作系统的 在新加坡的 设置密码…
  • 使用puttygen生成ssh公钥私钥，在digitalocean上设置自己的公钥（由于SSH密钥使用的是信息安全的随机块创建，需要点击对话框的空白区域随机生成数据，要一直点击，直到完成 。我一开始就一直傻傻地等，后来才看到英文提示）
  • 使用PuTTY创建SSH会话，连接服务器
    • 选择windows，输入刚开新建的虚拟机的IP，使用默认端口号22
    • ssh->auth 选择刚刚上传的公钥对应的私钥
    • 创建session。以后直接点击session就可以开启对应的远程连接

  • 远程到国外的服务器上，安装shadowsock服务器端

    • 安装用到的三条命令

    • 编写shadowsocks.json配置脚本

      修改这个文件/etc/shadowsocks.json

    • ssserver -c /etc/shadowsocks.json -d运行shadowsocks

下载shadowsocks客户端，根据刚刚填的shadowsocks.json中的相应信息进行填写。然后启用服务就ok了。
